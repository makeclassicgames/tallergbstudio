    Publicar nuestro juego

    Por último, vamos a publicar nuestro juego para poder compartirlo con otras personas. GBStudio nos permite exportar nuestro juego en diferentes formatos, como ROM para Game Boy, archivo HTML5 para jugar en el navegador.

    Recuerda que al exportarlo a ROM, podemos jugarlo en un emulador de Game Boy o en una consola real utilizando un cartucho flash. Al exportarlo a HTML5, podemos jugarlo directamente en el navegador web, lo que facilita compartirlo con otras personas sin necesidad de instalar ningún software adicional, e incluso publicarlo en plataformas como itch.io o Game Jolt.

    Para exportar nuestro juego, pulsaremos el botón de exportar en la parte superior derecha de la ventana de GBStudio, y seleccionaremos el formato de exportación que deseamos. Luego, seleccionaremos la carpeta donde queremos guardar el archivo exportado y pulsaremos guardar.

    Formatos de exportación

    GBStudio nos permite exportar nuestro juego en los siguientes formatos:

    • ROM: Archivo de juego para Game Boy, que se puede jugar en un emulador de Game Boy o en una consola real utilizando un cartucho flash.
    • HTML5: Archivo de juego para jugar en el navegador web, que se puede compartir fácilmente con otras personas sin necesidad de instalar ningún software adicional.
    • .pocket: Archivo compatible con la consola Analogue Pocket, que se puede jugar en la consola directamente sin necesidad de un cartucho flash.

    Una vez exportado nuestro juego, podemos probarlo en el formato que hayamos elegido, y compartirlo con otras personas para que puedan jugarlo y disfrutarlo.

    Con esto, hemos completado nuestro juego de plataformas para Game Boy, y hemos aprendido a utilizar GBStudio para crear un juego completo, desde la creación de escenas y actores, hasta la definición de acciones y eventos, y la exportación del juego en diferentes formatos. ¡Felicidades por tu logro!
