# Referencias

* [GBStudio](https://www.gbstudio.dev/) - Herramienta de desarrollo de juegos para Game Boy y Game Boy Color.
* [Game Boy (Wikipedia)](https://es.wikipedia.org/wiki/Game_Boy) - Información sobre la consola Game Boy y Game Boy Color.
* [Analogue Pocket](https://www.analogue.co/pocket) - Consola portátil que permite jugar a juegos de Game Boy y Game Boy Color, entre otros sistemas.
* [Kenney](https://kenney.nl/assets) - Recursos gráficos y de audio gratuitos para juegos, incluyendo sprites y tilesets.
* [Make Classic Games](https://makeclassicgames.dev/) - Sitio Web del Canal de YouTube y twitch "Make Classic Games", donde se pueden encontrar tutoriales y recursos para crear juegos retro.
