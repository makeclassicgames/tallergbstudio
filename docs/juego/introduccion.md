# Juego de Plataformas

Vamos a crear un juego de plataformas para Game Boy y Game Boy Color utilizando GBStudio.

Durante las siguientes secciones, aprenderás a diseñar niveles, personajes y eventos para tu juego de plataformas, así como a configurar las propiedades del proyecto y probar el juego en un emulador, e incluso exportarlo a un archivo binario que se pueda cargar en un cartucho físico o ejecutar en un emulador de Game Boy y Game Boy Color.

## Creación de un nuevo proyecto

Vamos a crear un nuevo proyecto en GBStudio para nuestro juego de plataformas. Para ello, puedes cerrar y volver a abrir GBStudio, o bien hacer click en _Archivo_ y luego en _Nuevo Proyecto_.

En esta ocasión, seleccionaremos la plantilla de _Proyecto vacío_ para poder crear nuestro juego desde cero y personalizarlo a nuestro gusto.

Una vez creado el proyecto, deberás ver la interfaz principal de GBStudio, donde podrás comenzar a diseñar tu juego de plataformas.
