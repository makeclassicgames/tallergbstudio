# Game Boy Color

La Game Boy Color es una versión mejorada de la Game Boy original, lanzada al mercado en 1998. Esta consola portátil de Nintendo introdujo varias mejoras significativas en comparación con su predecesora, incluyendo una pantalla a color, mayor capacidad de procesamiento y compatibilidad con los juegos de la Game Boy original. La Game Boy Color se convirtió en un éxito comercial y fue muy popular entre los jugadores de todas las edades, consolidando aún más la posición de Nintendo en el mercado de consolas portátiles.

## Características principales de la Game Boy Color

- **Pantalla a color:** La Game Boy Color cuenta con una pantalla a color de 2.3 pulgadas, lo que permite a los jugadores disfrutar de gráficos más vibrantes y detallados en comparación con la pantalla monocromática de la Game Boy original.
- **Compatibilidad con juegos de Game Boy:** La Game Boy Color es compatible con la mayoría de los juegos de la Game Boy original, lo que permite a los jugadores disfrutar de su biblioteca de juegos existente mientras exploran los nuevos títulos diseñados específicamente para la Game Boy Color.
- **Mayor capacidad de procesamiento:** La Game Boy Color cuenta con un procesador más rápido y una mayor capacidad de memoria en comparación con la Game Boy original, lo que permite a los desarrolladores crear juegos más complejos y con mejores gráficos.
- **Diseño compacto y portátil:** Al igual que la Game Boy original, la Game Boy Color mantiene un diseño compacto y portátil, lo que permite a los jugadores llevar su consola a cualquier lugar y disfrutar de sus juegos en movimiento.

### Arquitectura de la Game Boy Color

Veamos un resumen de la arquitectura de la Game Boy Color, que nos ayudará a comprender cómo funciona internamente y cómo se pueden desarrollar juegos para esta consola.

* **CPU:** La Game Boy Color utiliza un procesador Sharp LR35902, similar al de la Game Boy original, pero con una velocidad de reloj más alta de 8 MHz. Esto permite un rendimiento mejorado y la capacidad de ejecutar juegos más complejos.
* **Memoria:** La consola cuenta con 32 KB de RAM interna y 256 KB de ROM (ampliable usando multibanco) para almacenar los programas. Además, los cartuchos de juego pueden incluir memoria adicional para guardar datos de progreso y permitir características avanzadas en los juegos.
* **Pantalla:** La pantalla de la Game Boy Color es una pantalla LCD de 2.3 pulgadas con una resolución de 160x144 píxeles y una paleta de 32,768 colores. Esto permite a los desarrolladores crear gráficos más detallados y coloridos en comparación con la Game Boy original.
* **Audio:** La Game Boy Color mantiene el sistema de audio integrado de la Game Boy original, pero con mejoras en la calidad del sonido. Los desarrolladores pueden crear música y efectos de sonido más complejos utilizando el chip de audio de la consola.
